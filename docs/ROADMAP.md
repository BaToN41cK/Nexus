## Текущий статус

Nexus v1.0.0 включает:
- ✅ CLI с командами `run`, `interactive`, `search`, `history`, `status`, `cache-clear`, `update`, `test`, `debug`, `version`
- ✅ LLM-провайдеры: Groq, OpenAI, Anthropic, Ollama
- ✅ Web-поиск: DuckDuckGo, Tavily, SearXNG, Bing
- ✅ Загрузка контента: YouTube, PDF, DOCX, PPTX, Excel, веб-страницы
- ✅ Кэширование ответов с TTL
- ✅ Контекст диалога
- ✅ Rich UI: Markdown, подсветка кода, прогресс-бары
- ✅ ReAct-агент (Reasoning + Acting)
- ✅ MCP-сервер (Claude Desktop, Cursor, Continue)
- ✅ Подключаемая память (JSON, SQLite)
- ✅ Интернационализация (ru, en)
- ✅ Docker-образ

## Краткосрочные планы (1–3 месяца)

### Приоритет 1

- [ ] **Plugin System** — система плагинов для расширения функционала
- [ ] **Function Calling** — поддержка function calling у провайдеров (OpenAI, Anthropic)
- [ ] **Улучшенный ReAct** — streaming промежуточных шагов, визуализация в реальном времени

### Приоритет 2

- [ ] **Кастомные промпты** — пользовательские системные промпты из файлов
- [ ] **Экспорт истории** — экспорт в Markdown, HTML, PDF
- [ ] **Множественные провайдеры** — одновременное использование нескольких провайдеров

### Приоритет 3

- [ ] **TUI (Terminal UI)** — расширенный интерфейс с панелями, деревьями, таблицами
- [ ] **История версий промптов** — отслеживание изменений системного промпта

## Как предложить фичу

### GitHub Issues

1. Создайте Issue с тегом `enhancement`
2. Опишите:
   - Проблему, которую решает фича
   - Предлагаемое решение
   - Альтернативы
   - Приоритет (nice-to-have / important / critical)

### Discord / Telegram

- Присоединитесь к сообществу
- Обсудите идею
- Получите обратную связь

### Pull Request

Если вы хотите реализовать фичу:

1. Fork репозиторий
2. Создайте ветку `feature/my-feature`
3. Реализуйте фичу
4. Напишите тесты
5. Обновите документацию
6. Откройте PR
