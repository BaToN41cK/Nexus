# Nexus MCP Сервер

> 🔙 Назад → [README.md](README.md)

**Model Context Protocol (MCP)** — это стандарт для подключения ИИ-инструментов к LLM-клиентам. Nexus предоставляет MCP-сервер, который позволяет клиентам (Claude Desktop, Cursor, Continue и др.) использовать Nexus как инструмент для поиска, загрузки контента и генерации ответов.

## Возможности

- **`nexus_run`** — задать вопрос Nexus (использует настроенного LLM-провайдера)
- **`nexus_search`** — поиск в интернете через настроенный бэкенд (DuckDuckGo / Tavily / SearXNG / Bing)
- **`nexus_fetch`** — загрузить и извлечь текст из URL (веб-страница, PDF, DOCX, YouTube и др.)

---

## Требования

- Python 3.9+
- Установленный Nexus (`pip install -e .`)
- Настроенный API-ключ провайдера (Groq / OpenAI / Anthropic)
- Опционально: `pip install mcp` (SDK для MCP-сервера)

---

## Установка

### 1. Установка Nexus (если ещё не установлен)

```bash
pip install -e .
```

### 2. Установка MCP SDK (опционально)

```bash
pip install mcp
```

> MCP SDK не является обязательной зависимостью Nexus — требуется только для режима MCP. Если SDK не установлен, `nexus mcp` выведет понятное сообщение об ошибке.

### 3. Настройка API-ключа

Убедитесь, что API-ключ настроен в `~/.nexus/.env` или в переменной окружения:

```bash
export GROQ_API_KEY=your_key_here
```

---

## Запуск сервера

```bash
nexus mcp
```

Процесс будет блокирован, считывая JSON-RPC сообщения из stdin и записывая ответы в stdout. Это стандартный способ работы MCP-серверов.

> **Важно:** Сервер работает в режиме stdio и не принимает сетевые подключения. Для работы с ним 필요한 MCP-клиент, который запускает сервер как дочерний процесс.

## Настройка клиентов

### Claude Desktop

**macOS:** `~/Library/Application Support/Claude/claude_desktop_config.json`

**Windows:** `%APPDATA%\Claude\claude_desktop_config.json`

```json
{
  "mcpServers": {
    "nexus": {
      "command": "nexus",
      "args": ["mcp"],
      "env": {
        "GROQ_API_KEY": "ваш-ключ-здесь"
      }
    }
  }
}
```

Если Nexus установлен в виртуальном окружении, укажите полный путь:

```json
{
  "mcpServers": {
    "nexus": {
      "command": "/path/to/venv/bin/nexus",
      "args": ["mcp"],
      "env": {
        "GROQ_API_KEY": "ваш-ключ-здесь"
      }
    }
  }
}
```

Перезапустите Claude Desktop. Три инструмента `nexus_*` появятся в панели выбора инструментов.

### Cursor

**Файл:** `~/.cursor/mcp.json`

```json
{
  "mcpServers": {
    "nexus": {
      "command": "nexus",
      "args": ["mcp"]
    }
  }
}
```

Перезапустите Cursor. Nexus будет доступен через панель MCP.

### Continue

**Файл:** `~/.continue/config.json`

```json
{
  "mcpServers": [
    {
      "name": "nexus",
      "command": "nexus",
      "args": ["mcp"]
    }
  ]
}
```

### Другие клиенты

Любой MCP-клиент, поддерживающий серверы через stdio, подходит. Укажите путь к исполняемому файлу `nexus` с аргументом `mcp`.

## Инструменты

### `nexus_run`

Задать вопрос Nexus. Использует настроенного LLM-провайдера и может автоматически применять web-поиск (если он включён в конфиге).

**Схема входных данных:**

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| `prompt` | `string` | ✅ | Вопрос или задание пользователя |
| `system_prompt` | `string` | ❌ | Опциональная системная инструкция |

**Пример вызова:**

```json
{
  "prompt": "Что такое Model Context Protocol?",
  "system_prompt": "Отвечай кратко на русском языке"
}
```

**Ответ:** Текстовый ответ от LLM.

### `nexus_search`

Поиск в интернете через настроенный бэкенд (DuckDuckGo / Tavily / SearXNG / Bing).

**Схема входных данных:**

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| `query` | `string` | ✅ | Поисковый запрос |
| `max_results` | `integer` | ❌ | Максимум результатов (1–20, по умолчанию 5) |

**Пример вызова:**

```json
{
  "query": "последние новости Python 3.13",
  "max_results": 5
}
```

**Ответ:** Форматированный список результатов:
```
- Title | URL | Snippet
- Title 2 | URL 2 | Snippet 2
```

### `nexus_fetch`

Загрузить и извлечь текст из URL. Поддерживаются: веб-страницы, PDF, DOCX, PPTX, Excel, YouTube и др.

**Схема входных данных:**

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| `url` | `string` | ✅ | URL для загрузки |

**Пример вызова:**

```json
{
  "url": "https://example.com/article"
}
```

**Ответ:** Извлечённый текст (до 12 000 символов). Если текст больше — обрезается с уведомлением.

## Примеры использования

### Claude Desktop

После настройки MCP-сервера вы можете задавать вопросы прямо в Claude Desktop:

```
Пользователь: Найди последние новости о Python
Claude: [вызывает nexus_search] ...
Пользователь: Загрузи эту статью и сделай саммари
Claude: [вызывает nexus_fetch] ... [вызывает nexus_run] ...
```

### Программное использование

```python
import subprocess
import json

# Запуск MCP-сервера и отправка запроса
# (упрощённый пример — на практике используйте MCP SDK)
result = subprocess.run(
    ["nexus", "run", "Что такое MCP?"],
    capture_output=True,
    text=True
)
print(result.stdout)
```

## Безопасность

- MCP-сервер работает через **stdio** — он не принимает сетевые подключения
- API-ключи передаются через переменные окружения, а не через аргументы командной строки
- Сервер не логирует запросы и ответы (при стандартной конфигурации)
- Рекомендуется использовать API-ключи с ограниченными правами

## Troubleshooting

### Требуется пакет 'mcp'

**Симптом:** `The 'mcp' package is required to run the Nexus MCP server.`

**Решение:**
```bash
pip install mcp
```

### Неверная конфигурация Nexus

**Симптом:** `Invalid Nexus configuration: ...`

**Решение:**
1. Проверьте файл `~/.nexus/config.yaml`
2. Пересоздайте его, выполнив любую команду nexus:
   ```bash
   nexus status
   ```

### Нет результатов от nexus_search

**Симптом:** Поиск возвращает пустые результаты.

**Решение:**
1. Включите web-поиск в конфиге:
   ```yaml
   web_search:
     enabled: true
   ```
2. Проверьте доступность бэкенда:
   ```bash
   nexus search "тест" --max 1
   ```

### Claude Desktop не видит инструменты

**Решение:**
1. Убедитесь, что путь к `nexus` правильный
2. Перезапустите Claude Desktop
3. Проверьте JSON-синтаксис конфигурационного файла
4. Посмотрите логи Claude Desktop на наличие ошибок

### Oшибка 'No API key configured'

**Симптом:** `[Nexus MCP] No API key configured for the active provider.`

**Решение:**
Установите переменную окружения в конфиге MCP-клиента:
```json
{
  "env": {
    "GROQ_API_KEY": "ваш-ключ"
  }
}
